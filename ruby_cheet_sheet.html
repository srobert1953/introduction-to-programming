Strings

  • String is a sequence of characters.
  • Characters are ordered so each character has its own index.
  • Use single or double quotations '', "" to create a string. Using double quotations, interpolation can be used "#{}", but not with single quotations.
Returns 6th character from string

"This is code"[5]

=> "i"

Interpolation example

a = 53

puts "My favorite number is #{a}"

My favorite number is 53

=> nil

Ineresting classes

#<< Append a string with another string.
"Hello" << " World"
=> "Hello World"
#capitalize Capitalize the string, make the first letter capital. If all letters are capital, only first one will be capital.
"hello".capitalize
=> "Hello"
#chars Returns an array of characters in string. Same as "sttring".each_char.to_a .
"name".chars
=> ["n", "a", "m", "e"]
#chomp Removes a line separator from the end of line if one exists.
"line\n".chomp
=> "line"
#clear Makes the string empty. Destructive

a = "remove me"

a.clear

=> ""
#downcase Returns a string in lowercase if any character is uppercase.
"I am UPPERCASE"
=> "i am uppercase"
#empty? Returns true if the given string is empty.
"".empty?
=> true
#include? Returns true if string contains given character or string.
"Hello".include? "lo"
=> true
#length Gives length of a string.
"how long am I?".length
=> 14
#replace Replaces the content of string with another string. Destructive

a = "replace me"

a.replace "with this"

=> "with this"
#reverse Returns new string with characters in reverse order.
"string".reverse
=> "gnirts"
#strip Returns string with leading and trailing whitespace removed.
"    remove the empty space    ".strip
=> "remove the empty space"
#upcase Returns a string in uppercase if any character is lowercase.
"I am lowercase"
=> "I AM LOWERCASE"
