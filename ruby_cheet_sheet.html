Strings

  • String is a sequence of characters.
  • Characters are ordered so each character has its own index.
  • Use single or double quotations '', "". Using double quotations, interpolation can be used #{}, but not with single quotations.
Displaying 6th character from string

"This is code"[5]

=> "i"

Interpolation example

a = 53

puts "My favorite number is #{a}"

My favorite number is 53

=> nil

Type conversion

  • Convert one type to another type
Convert string to integer

"3".to_i

Convert string to float

"4".to_f
